# spec-dev-sandbox

Unityプロジェクトのサンドボックス環境

## プロジェクト概要

ゲームビューでボタンを押すと「Hello World」とログに書き出される機能を実装するプロジェクトです。

## 開発環境

- Unity Engine
- C#

## セットアップ

1. Unityエディタでプロジェクトを開く
2. メニューから `Tools > Setup Hello World Scene` を選択してCanvasとButtonを自動生成
3. シーンを開く
4. ゲームビューでボタンをクリックして動作確認

## セットアップ手順（詳細）

### CanvasとButtonの配置

Unityエディタで以下の手順を実行してください：

1. Unityエディタを開く
2. メニューバーから `Tools > Setup Hello World Scene` を選択
3. CanvasとButtonが自動的に作成されます
4. シーンを保存（Ctrl+S）

または、手動で以下の手順を実行することもできます：

1. Hierarchyで右クリック → UI → Canvas
2. Canvasを選択した状態で右クリック → UI → Legacy → Button
3. Buttonの位置を画面中央に調整
